# Contributing

This is an active investigation. Contributions are welcome from researchers, journalists, and legal professionals.

## How to Contribute

1. Fork this repository
2. Submit findings via Pull Request with specific EFTA/SDNY document citations
3. All claims must be verifiable against publicly available DOJ releases

## Contact

For sensitive contributions or to request collaborator access, contact the repository owner through secure channels.

## Guidelines

- All findings must cite specific document numbers
- No doxxing of victims or witnesses
- No unverified speculation presented as fact
- Distinguish clearly between documented evidence and inference
- Protect victim privacy above all else

## Collaborators

Collaborator identities are kept private for security purposes.
